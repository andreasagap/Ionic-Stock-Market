## Don't forget to change the firebase's script
```
<script>
      // Initialize Firebase
      var config = {
        apiKey: "---Your ApiKey---",
        authDomain: "---Your authDomain---",
        databaseURL: "---Your databaseURL---",
        projectId: "---projectId---",
        storageBucket: "---storageBucket---",
        messagingSenderId: "---MessagingSenderId---"
      };
      firebase.initializeApp(config);
</script>
```
